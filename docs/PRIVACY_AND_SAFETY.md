# Privacy And Safety

This project is local-first, but local files can still contain sensitive information if you add it. Do not commit local configs, generated reports, generated resumes, SQLite databases, profile exports, or application history.

Do not store sensitive identity, government ID, exact birth date, financial account, health, demographic, veteran, or disability information in this repository. Answer those fields manually in the employer application system.

This project must not be used for auto-apply, auto-submit, login automation, captcha bypass, Cloudflare bypass, proxy evasion, or private data scraping. It is a local search, scoring, reporting, and manual application-assist workflow.

Before publishing, run tests, run the privacy grep from the release checklist, inspect tracked files, and do a manual review.
